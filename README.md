## 🚀 How to Run

### Option 1 — Direct Browser Open
```bash
# Simply double-click index.html
# OR right-click → Open with → Chrome / Firefox
```

### Option 2 — Live Server (VS Code)
```bash
# Install Live Server extension in VS Code
# Right-click index.html → "Open with Live Server"
```

### Option 3 — Python Local Server
```bash
cd pragati-sahayak/
python -m http.server 8000
# Open: http://localhost:8000
```

### Option 4 — Deploy to GitHub Pages
```bash
git init
git add .
git commit -m "🚀 Launch Pragati Sahayak website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/pragati-sahayak.git
git push -u origin main
# Enable GitHub Pages in repo Settings → Pages → Branch: main
```

## 🔧 Customization Guide

### Change Phone Number
Search & replace `7253985468` with your number in the HTML file.

### Change UPI ID
Search `7253985468@ybl` and update to your UPI ID.

### Change Email
Search `pragatisahayak@gmail.com` and update.

### Add New Portfolio Item
Find `portfolioItems` array in the `<script>` section and add:
```javascript
{
    title: "Your Project Name",
    category: "Category",
    image: "https://your-image-url.com/img.jpg",
    metric: "Result metric"
}
```

### Change Pricing
Find the `PLANS` array in `<script>` and update:
```javascript
const PLANS = [
    { name: 'Fast',    price: 12999, subtitle: '...' },
    { name: 'Growth',  price: 16999, subtitle: '...' },
    { name: 'Premium', price: 27999, subtitle: '...' }
]
```
